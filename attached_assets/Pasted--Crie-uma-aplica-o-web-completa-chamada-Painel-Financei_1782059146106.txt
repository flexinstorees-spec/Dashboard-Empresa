
Crie uma aplicação web completa chamada Painel Financeiro, focada em consolidar os dados financeiros de todas as minhas ofertas em uma única visão.

O principal objetivo é resolver uma limitação da UTMify: atualmente cada oferta possui seus próprios dados, mas não existe uma visão consolidada da empresa inteira.

O sistema deve conectar-se ao MCP da UTMify, buscar os dados reais e gerar automaticamente métricas globais do negócio.

REQUISITOS DA INTEGRAÇÃO

O sistema deve:

* Conectar automaticamente ao MCP
* Descobrir e utilizar as ferramentas disponíveis
* Buscar dados reais diretamente do MCP
* Não utilizar dados fictícios ou simulados quando houver conexão disponível
* Atualizar os dados automaticamente
* Tratar erros de conexão
* Exibir status da sincronização
* Salvar histórico das métricas no banco de dados
* Detectar automaticamente novas ofertas adicionadas futuramente sem necessidade de alterar código

Todas essas métricas devem considerar a soma de todas as ofertas encontradas no MCP.

CÁLCULOS

Sempre que os dados estiverem disponíveis no MCP:

Receita Total = soma da receita de todas as ofertas

Lucro Total = soma do lucro de todas as ofertas

ROI Geral = lucro total ÷ gasto total

Ticket Médio = receita total ÷ quantidade de vendas

Todos os demais cálculos devem utilizar os dados reais retornados pelo MCP.

Taxas, reembolsos e demais métricas devem ser importados automaticamente quando estiverem disponíveis.
